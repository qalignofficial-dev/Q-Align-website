# Q-Align (큐얼라인)

Q-Align (큐얼라인) is a daily alignment OS that helps teams stay healthy and aligned.

## Overview
Q-Align(큐얼라인)은 매일의 질문으로 팀의 기준과 목표를 정렬해, 번아웃과 비용 누수를 줄이는 팀 상태 OS입니다.

### Core Values
1. 팀이 흔들리지 않으려면 방향이 매일 같아야 해요.
2. 감으로 팀 상태를 추측하지 마세요.
3. 기준이 어긋나면 실행은 느려집니다.
4. 보이지 않는 팀의 정렬을 매일 확인하세요.
5. 오늘 중요한 게, 모두에게 같나요?
